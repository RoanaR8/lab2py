Buat Repository untuk lab2py

pertama-tama login terlebih dahulu ke github jika sudah memiliki akun tinggal login jika belum bikin akun terlebih dahulu

kemudian buat directory bernama lab2py dan kemudian centang add a readme file

setelah itu buka git dan ketikan pwd

dan cek directory yg disimpan didata mana menempatkannya

ketikkan cd /d/Latihan/

kemudian ketikkan ls

kemudian ketikkan ls -1 maka tampilannya masih kosong

Buat file README.md

menambahkan git clone untuk menambahkan file readme dan copas url repositorynya

ketikkan git clone https://github.com/RoanaR8/lab2py.git

Tambahkan semua file pada latihan

letakan file readme dan lab2py.html ke folder latihan

kemudian lakukan pengcodingan di vscode untuk mendekorasi file latihan_lab2py.html

Commit dan Push repository ke Github

buka git

ketikkan cd lab2py

kemudian git add latihan_lab2py.html

git status

tambahkan commit

ketikkan git commit -m "menambahkan file latihan_lab2py" untuk namanya bebas mau diisi apa saja terserah user

kemudian ketikkan git status lagi

kemudian push ketikkan git push -u origin main
